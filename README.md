# Studio Landing Page

A minimal, high-performance landing page for a software engineering studio specializing in internal tools and custom dashboards. Built with vanilla tech and zero dependencies for maximum speed.

## ✨ Features

* **Adaptive Themes:** Full light/dark mode support using unified CSS variables.
* **Dynamic Hero Toggler:** Smooth, non-shifting text animation transitioning between studio focus areas.
* **Zero Dependencies:** Pure HTML, CSS, and Vanilla JS for instant load times.

## 🛠️ Tech Stack

* **Font:** Plus Jakarta Sans
* **Styles:** Custom CSS properties (Slate-based theme)
* **Scripts:** Vanilla JS (ES6+)

## 🚀 Quick Start

Since this project uses native web standards, no installation or npm setup is required.

1. Clone the repository.
2. Open index.html directly in any modern web browser.
